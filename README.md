# Kronos Keeper

Kronos Keeper is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It focuses on detecting file corruption and monitoring system integrity to ensure long-term reliability of your Kodi installation.

---

## Features
- **Integrity Scans** – Performs scheduled checks for corrupted or missing files. 
- **Hash Verification** – Compares file signatures to detect silent corruption. 
- **Filesystem Monitoring** – Identifies errors or inconsistencies that may affect performance. 
- **Smart Scheduling** – Runs only on safe intervals (e.g., idle on homepage, weekly checks). 
- **Status Notifications** – Reports scan results with clear outcomes and next steps. 
- **Lightweight Service** – Runs silently in the background with minimal resource usage.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Python 3 (bundled with Kodi)
